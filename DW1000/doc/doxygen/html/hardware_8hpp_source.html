112 //
113 // Specify a bitmask that defines optional properties of the NIC.
114 // This miniport indicates receive with NdisMIndicateReceiveNetBufferLists
115 // function. Such a driver should set this NDIS_MAC_OPTION_TRANSFERS_NOT_PEND
116 // flag.
117 //
118 // NDIS_MAC_OPTION_NO_LOOPBACK tells NDIS that NIC has no internal
119 // loopback support so NDIS will manage loopbacks on behalf of
120 // this driver.
121 //
122 // NDIS_MAC_OPTION_COPY_LOOKAHEAD_DATA tells the protocol that
123 // our receive buffer is not on a device-specific card. If
124 // NDIS_MAC_OPTION_COPY_LOOKAHEAD_DATA is not set, multi-buffer
125 // indications are copied to a single flat buffer.
126 //
