Detailed Description

EMSK Designware GPIO Objects.

Implement the EMSK board GPIO object using the DesignWare GPIO device driver. Only need to implement some DesignWare GPIO structures and combine with EMSK GPIO hardware resources, like in C++ class instantiation. For example, in C++, define a class called gpio class gpio {

}; then instantiate the gpio class: gpio emsk_gpio_a;
