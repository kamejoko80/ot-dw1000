This method encodes a single byte into a buffer at aIterator.

If there is no space to add the byte, the write iterator remains the same.

Parameters

Return values

This method encodes the frame into a buffer at aIterator.

This method returns success only if there is space in buffer to encode the entire frame. If there is no space to encode the entire frame, the write iterator remains the same.

[in]aInLengthThe number of bytes in aInBuf to encode.
