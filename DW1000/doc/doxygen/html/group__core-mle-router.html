ThreadError SetPreferredRouterId ( uint8_t  aRouterId)

This method sets the preferred Router Id. Upon becoming a router/leader the node attempts to use this Router Id. If the preferred Router Id is not set or if it can not be used, a randomly generated router Id is picked. This property can be set when he device role is detached or disabled.

Return values
kThreadError_NoneSuccessfully set the preferred Router Id.
kThreadError_InvalidStateCould not set (role is other than detached and disabled)
