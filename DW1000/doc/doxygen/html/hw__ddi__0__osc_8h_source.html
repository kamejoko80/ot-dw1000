185 // Field: [10] XOSC_LF_DIG_BYPASS
186 //
187 // Bypass XOSC_LF and use the digital input clock from AON for the xosc_lf
188 // clock..
189 //
190 // 0: Use 32kHz XOSC as xosc_lf clock source
191 // 1: Use digital input (from AON) as xosc_lf clock source.
192 //
193 // This bit will only have effect when SCLK_LF_SRC_SEL is selecting the xosc_lf
194 // as the sclk_lf source. The muxing performed by this bit is not glitch free.
195 // The following procedure should be followed when changing this field to avoid
196 // glitches on sclk_lf..
