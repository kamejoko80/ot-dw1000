◆ OutboundFrameFeedMessage()

ThreadError OutboundFrameFeedMessage ( otMessage  aMessage)
virtual

This method adds a message to the current outbound frame being written.

If no buffer space is available, this method will discard and clear the frame before returning an error status. In case of success, the passed-in message will be owned by outbound buffer and will be freed when either the the frame is successfully sent and removed or if the frame is discarded.

Parameters
[in]aMessageA message instance to be added to current frame.
Return values
kThreadError_NoneSuccessfully added the message to the frame.
kThreadError_NoBufsInsufficient buffer space available to add message.
