QSPI_SECTION __attribute__ ( (unused)  )

Get Device ID when Flash is not in Power Down mode.

Read the JEDEC manufacturer ID, device type and device density using command 0x9F.

Erase a sector of the Flash.

Write the Status Register 1 of the Flash.

Returns
uint8_t The Device ID of the Flash
Note
The function blocks until the Flash executes the command.
Parameters
[in]valueThe value to be written.
Note
This function blocks until the Flash has processed the command. No verification that the value has been actually written is done though. It is up to the caller to decide whether such verification is needed or not and execute it on its own.
Parameters
[in]addrThe address of the sector to be erased.
Note
This function blocks until the Flash has processed the command.
Parameters
[in]manufacturer_idPointer to the variable where the manufacturer ID will be returned
[in]device_typePointer to the variable where the device type will be returned
[in]densityPointer to the variable where the device density will be returned Configure dummy bytes in QSPI controller
[in]countNumber of dummy bytes (not including extra byte)
