void StandaloneRomCryptoInit ( void  )

Simplified initialization of the common ROM-RAM system needed prior to using the RomEncryption functions.

This simplified initialization shall not be used if using the stack software. The stack software implies calling the function CommonROM_Init() replacing this simplified method.

Note
SRAM area 0x20004F2C-0x20004FFF will be allocated (COMMON_RAM_BASE_ADDR region).
