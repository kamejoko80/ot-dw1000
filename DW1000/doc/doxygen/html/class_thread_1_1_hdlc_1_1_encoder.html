This method encodes a single byte into a buffer at aIterator.

If there is no space to add the byte, the write iterator remains the same.

Parameters
[in]aInByteA byte to encode and add.
[in,out]aIteratorA reference to a write buffer iterator. On successful exit, the iterator is updated.
Return values
kThreadError_NoneSuccessfully encoded and added the byte.
kThreadError_NoBufsInsufficient buffer space available to encode and add the byte.

uint16_t  aInLength,
BufferWriteIterator &  aIterator

This method encodes the frame into a buffer at aIterator.

This method returns success only if there is space in buffer to encode the entire frame. If there is no space to encode the entire frame, the write iterator remains the same.

Parameters
[in]aInBufA pointer to the input buffer.
[in]aInLengthThe number of bytes in aInBuf to encode.
[in,out]aIteratorA reference to a write buffer iterator. On successful exit, the iterator is updated.
Return values
kThreadError_NoneSuccessfully encoded the HDLC frame.
kThreadError_NoBufsInsufficient buffer space available to encode the HDLC frame.
