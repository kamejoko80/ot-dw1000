Table of pflash sizes.

The index into this table is the value of the SIM_FCFG1.PFSIZE bitfield.

The values in this table have been right shifted 10 bits so that they will all fit within an 16-bit integer. To get the actual flash density, you must left shift the looked up value by 10 bits.
