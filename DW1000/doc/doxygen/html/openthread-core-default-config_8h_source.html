764 /*
765  * @def OPENTHREAD_CONFIG_CHILD_SUPERVISION_INTERVAL
766  *
767  * The default supervision interval in seconds used by parent. Set to zero to disable the supervision process on the
768  * parent.
769  *
770  * Applicable only if child supervision feature is enabled (i.e., `OPENTHREAD_ENABLE_CHILD_SUPERVISION ` is set).
771  *
772  * Child supervision feature provides a mechanism for parent to ensure that a message is sent to each sleepy child
773  * within the supervision interval. If there is no transmission to the child within the supervision interval, child
774  * supervisor will enqueue and send a supervision message (a data message with empty payload) to the child.
775  *
776  */
