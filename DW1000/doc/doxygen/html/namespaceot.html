NCP Buffer Fuzz testing

Randomly decide if to read or write a frame to the NCP buffer (use kReadProbability in percent to control the behavior).

When writing a frame, use a random length (1 up to kMaxFrameLen) and generate random byte sequences. When reading a frame ensure the length and the content matches what was written earlier. Handle the cases where buffer gets full or empty.
