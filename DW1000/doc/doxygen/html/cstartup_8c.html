Detailed Description

cstartup for Dialog da15000. Called by __BOOT_STARTUP to initialize the processor on boot. The caller is aliased in tool.h to a compiler dependant value: IAR: __low_level_init GCC: __gcc_program_start

For Cortex-M cores, called from cpu/arm/cortex-m/boot/cstartup.c.
