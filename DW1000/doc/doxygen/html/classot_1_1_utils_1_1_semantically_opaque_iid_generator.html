Detailed Description

This class implements the Method for Generating Semantically Opaque IIDs with IPv6 SLAAC (RFC 7217).

otError CreateIid ( otInstance *  aInstance,
otNetifAddress *  aAddress 
)

This method creates semantically opaque IID for given IPv6 address and context.

The generator starts with DAD counter provided as a class member field. The DAD counter is automatically incremented at most kMaxRetries times if creation of valid IPv6 address fails.

Parameters
[in]aInstanceA pointer to an OpenThread instance.
[in,out]aAddressA pointer to structure containing IPv6 address for which IID is being created.
Return values
OT_ERROR_NONEGenerated valid IID.
OT_ERROR_INVALID_ARGSAny given parameter is invalid.
OT_ERROR_IP6_ADDRESS_CREATION_FAILURECould not generate IID due to RFC 7217 restrictions.
