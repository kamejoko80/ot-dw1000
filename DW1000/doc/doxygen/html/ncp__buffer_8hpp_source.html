278  /*
279  * NcpFrameBuffer Implementation
280  * -----------------------------
281  *
282  * NcpFrameBuffer internally stores a frame as a sequence of data segments. The data segments are stored in the
283  * the main buffer `mBuffer`. mBuffer is utilized as a circular buffer.
284 
285  * Messages (which are added using `InFrameFeedMessaged()`) are not copied in the `mBuffer` but instead are
286  * enqueued in a message queue `mMessageQueue`.
287  *
288  * The data segments include a header before the data portion. The header is 2 bytes long is formated as follows
289  *
290  * Bit 0-13: Give the length of the data segment (max segment len is 2^14 = 16,384 bytes).
291  * Bit 14: Flag bit set to indicate that this segment has an associated `Message` (appended to its end).
292  * Bit 15: Flag bit set to indicate that this segment defines the start of a new frame.
293  *
294  * Bit 15 Bit 14 Bits: 0 - 13
295  * +--------------+--------------+--------------------------------------------------------+
296  * | New Frame | Has Message | Length of segment (excluding the header) |
297  * +--------------+--------------+--------------------------------------------------------+
298  *
299  * The header is encoded in big-endian (msb first) style.
300 
301  * Consider the following calls to create a frame:
302  *
303  * ncpBuffer.InFrameBegin();
304  * ncpBuffer.InFrameFeedData("Hello", 5);
305  * ncpBuffer.InFrameFeedData("There", 5);
306  * ncpBuffer.InFrameFeedMessage(*someMessage);
307  * ncpBuffer.InFrameFeedData("Bye", 3);
308  * ncpBuffer.InFrameEnd();
309  *
310  * This frame is stored as two segments:
311  *
312  * - Segment #1 contains "HelloThere" with a header of `0xC00A` which shows that this segment contains 10 data
313  * bytes, and it starts a new frame, and also must include a message from the message queue.
314  *
315  * - Segment #2 contains "Bye" with a header value of `0x0003` showing length of 3 and no appended message.
316  *
317  * +-----+-----+-----+-----+-----+-----+-----+-----+-----+-----+-----+-----+-----+-----+-----+-----+-----+
318  * | C0 | 0A | `H` | 'e' | 'l' | 'l' | 'o' | 'T' | 'h' | 'e' | 'r' | 'e' | 00 | 03 | 'B' | 'y' | 'e' |
319  * +-----+-----+-----+-----+-----+-----+-----+-----+-----+-----+-----+-----+-----+-----+-----+-----+-----+
320  * \ / \ /
321  * Segment #1 Header Segment #2 Header
322  *
323  */
